# Final Project Struktur Data
## Aplikasi Manajemen Inventaris Barang

### Deskripsi
Aplikasi Manajemen Inventaris Barang adalah program berbasis Python yang digunakan untuk mengelola data barang dan transaksi inventaris. Data disimpan menggunakan file CSV sebagai media penyimpanan, dengan implementasi struktur data Hash Map dan Stack.

### Struktur Data yang Digunakan
- Hash Map (Dictionary) untuk menyimpan dan mengelola data barang.
- Stack untuk menyimpan riwayat transaksi.

### Fitur Program
- Menambah data barang
- Menampilkan data barang
- Mengubah data barang
- Menghapus data barang
- Mencatat transaksi barang
- Menampilkan riwayat transaksi
- Penyimpanan data menggunakan file CSV

### File Project
- `tugas_akhir.py` → Program utama
- `barang.csv` → Data barang
- `transaksi.csv` → Data transaksi
- `diagramstrukdat.png` → Flowchart program

### Cara Menjalankan Program
1. Pastikan Python telah terinstal.
2. Jalankan file program:

```bash
python tugas_akhir.py
```

### Teknologi yang Digunakan
- Python
- CSV
- Hash Map
- Stack
